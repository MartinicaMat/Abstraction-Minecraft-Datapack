# Abstraction

An efficient datapack designed to give players overpowered custom items instantly. Fully compatible with modern version formats, this pack uses a robust loot-table execution method to bypass recent command syntax limitations.

### Key Features
* **The Potion of the Gods:** A custom potion granting massive, long-lasting status effects (*Jump Boost*, *Health Boost*, and *Luck*).
* **The Sword of Ultimate Doom:** A devastating Netherite sword featuring custom lore, a bold purple name, and extreme enchantments to one-shot any boss.

Once the datapack is loaded into your world, simply run the following command in your chat:
```text
/function abstraction:give_potion
